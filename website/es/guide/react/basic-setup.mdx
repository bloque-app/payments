## ConfiguraciĂ³n del Backend

Crea una sesiĂ³n de checkout en tu backend usando `@bloque/payments`:

```typescript
import { Bloque } from '@bloque/payments';
import express from 'express';

const app = express();
app.use(express.json());

const bloque = new Bloque({
  secretKey: process.env.BLOQUE_SECRET_KEY!,
  mode: 'production',
});

app.post('/api/create-checkout', async (req, res) => {
  try {
    const checkout = await bloque.checkout.create({
      name: 'Mi Producto',
      items: [{ name: 'Producto', amount: 2999, quantity: 1 }],
      success_url: 'https://tuapp.com/success',
    });

    res.json({
      id: checkout.id,
      clientSecret: checkout.client_secret,
    });
  } catch (error) {
    console.error('Error creando checkout:', error);
    res.status(500).json({ error: error.message });
  }
});

app.listen(3000);
```

### `AppearanceConfig`


### `PaymentResult`


### `PaymentMethod`


| Valor | DescripciĂ³n |
